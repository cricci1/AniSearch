# AniSearch

![image](./public/logo.png)

AniSearch is a Anime Database Website created using NextJS, React, Bootstrap, CSS and the [Jikan API](https://jikan.moe/).

Users can search for their favorite Anime, Manga, Characters or People and get detailed information about them. 

This website is a **WORK IN PROGRESS**. The design and features are subject to change. Keep an eye out on the issues tab in order to see what's being work on and what is getting fixed.
